# 🌍 马吉的平行世界 — 项目愿景

> “The Landlord’s Game was never about becoming rich. It was about showing how rent enriches the few and impoverishes the many.”  
> — Lizzie Magie, 1906

## 🎯 我们为何而做？

Lizzie Magie 在 1904 年发明了《地主游戏》（The Landlord’s Game），其初衷是**批判土地垄断、倡导经济公平**。然而，这一思想在 1930 年代被商业化的“大富翁”（Monopoly）彻底掩盖。

“马吉的平行世界”旨在：
- **还原历史真相**：呈现马吉原版的“繁荣模式”（Prosperity Mode）
- **教育公众**：通过游戏体验理解垄断 vs. 合作的经济后果
- **开源共创**：让全球社区共同维护这一公共思想遗产
- **赋能教学**：为经济学、历史、游戏设计课程提供活教材

## 🌐 什么是“平行世界”？

本项目提供**两个平行规则世界**：
1. **垄断世界**（Monopoly）：赢家通吃，财富集中 → 展示问题  
2. **繁荣世界**（Prosperity）：土地收益共享，共同富裕 → 提供解法

玩家可在同一棋盘上切换视角，亲历“制度如何塑造结果”。

## 🧩 核心原则

- **开源免费**：MIT + CC BY-SA 4.0，允许教学、修改、再分发  
- **教育优先**：设计简洁、逻辑透明，适合课堂使用  
- **社区驱动**：人人可贡献，从代码到内容  
- **历史尊重**：所有历史资料标注来源，马吉专利已进入公有领域

## 🚀 长期愿景

- 成为开源游戏标杆项目  
- 被全球 100+ 教育机构采用  
- 激发对经济制度的公共讨论  
- 启发更多“以游戏为媒介的社会实验”

**我们不是在做游戏，我们在重建被遗忘的公共思想。**  
加入我们，一起把 Lizzie Magie 的遗产归还给世界。
