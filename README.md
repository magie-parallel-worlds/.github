# 🤝 如何为“马吉的平行世界”做贡献

欢迎！无论你是开发者、设计师、教师、翻译者还是历史爱好者，你都可以在这里留下印记。

## 🌍 项目使命
还原 Lizzie Magie 被遗忘的经济公平思想，用开源游戏推动公众对垄断、合作与共同富裕的思考。

## 👥 贡献方式

### 1. **开发者**
- 实现功能模块（见 [模块架构](https://github.com/magie-parallel-worlds/.github/blob/main/ARCHITECTURE.md)）
- 修复 bug
- 编写测试（目标覆盖率 >70%）
- 优化性能

### 2. **设计师**
- UI/UX 设计（Figma 协作）
- 棋子/地图/图标创作
- 主题系统（复古、现代、暗黑等）

### 3. **内容创作者**
- 撰写 Lizzie Magie 传记
- 制作经济学解释内容
- 编写新手教程、教师指南

### 4. **翻译者**
- 翻译 README、UI、教育内容
- 支持目标：10+ 语言（已有中文、将推英文）

### 5. **测试者**
- 参与 Alpha/Beta 测试
- 提交清晰的 bug 报告
- 验证边缘 case（如财产清零、极端年龄）

### 6. **传播者**
- 制作视频、写博客
- 在学校/社群推广
- 组织游戏活动

## 🔧 贡献流程
1. **Fork** 目标仓库
2. 创建分支（如 `feat/new-ai-strategy` 或 `docs/translate-fr`）
3. 提交代码/内容
4. 确保通过 CI（如有）
5. 提交 **Pull Request**
6. 等待 Review（通常 3-5 天）

> 💡 小改动（如错别字）可直接提 PR；大功能建议先开 Issue 讨论。
